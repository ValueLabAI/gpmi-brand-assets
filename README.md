# GPMI Brand Assets

Static brand assets for Great Pacific Mortgage & Investments Ltd., served via jsDelivr CDN for use in the Content Engine plugin.

## Usage

Reference any file using jsDelivr:

```
https://cdn.jsdelivr.net/gh/ValueLabAI/gpmi-brand-assets@main/[folder]/[filename]
```

### Examples

```
https://cdn.jsdelivr.net/gh/ValueLabAI/gpmi-brand-assets@main/logos/logo-horizontal-white.png
https://cdn.jsdelivr.net/gh/ValueLabAI/gpmi-brand-assets@main/logos/logo-horizontal.webp
https://cdn.jsdelivr.net/gh/ValueLabAI/gpmi-brand-assets@main/fonts/GreatPacificLatin-Bold.woff2
https://cdn.jsdelivr.net/gh/ValueLabAI/gpmi-brand-assets@main/photography/GP-Hero-1.webp
```

## Folder structure

| Folder | Contents |
|---|---|
| `fonts/` | GreatPacificLatin woff2 files (all weights) |
| `logos/` | Logo variants: horizontal, vertical, favicon |
| `photography/` | Brand photography, team headshots, BC landscapes |
